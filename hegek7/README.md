## test.swf
	http://cdn.hegek7.com/test.swf

作用：通过利用 Flash Player 更新 UI 的接口，判断广告是否在可见区域。
实现机制：通过监听 Event.ENTER_FRAME 事件, 当 400ms 内 Flash Player 未更新UI，则认为广告未处于曝光区域。
代码解析：参见 hegek7/test.swf(as)/Scripts/ActionScript 3.0.as
